Pipeline

Download video + subtitle from youtube (youtube-dl)

convert subtitle in vtt to srt

run the subtitles through a forced aligner (gentle)

Do segmentation

  • Vision
    • off the shelf shot change detection (ML)
    • more primitive vision like histogram + edge
  • Audio Transcript
    • each sentence gets a segment
    • sequential topic models
    • maybe some other things
  • Time
    • equal time, but how many segments?
  • User interaction
    • where users want to go to
    • what users want to do
